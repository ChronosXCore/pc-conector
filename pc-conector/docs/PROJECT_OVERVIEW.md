# PC Conector - Project Overview

## 📋 Descripción General

**PC Conector** es una aplicación de escritorio multiplataforma diseñada para conectar dos PCs a través de una red WiFi local, permitiendo:

- **Compartir Portapapeles**: Copia en un PC, pega en el otro.
- **Compartir Mouse y Teclado**: Usa el mouse/teclado de un PC en el otro (KVM over IP).
- **Mouse a través de pantallas**: Mueve el mouse de un PC a otro como si fuera un segundo monitor.
- **Compartir Audio**: Transmite micrófono y altavoces entre PCs.

## 🎯 Propósito

Crear una herramienta gratuita, open-source, fácil de usar e intuitiva que reemplace soluciones comerciales como Synergy, Mouse Without Borders, o ShareMouse, añadiendo capacidades de audio que estas no tienen.

## 🔑 Características Principales

- Descubrimiento automático de PCs en la red local
- Conexión automática al iniciar el sistema
- Interfaz drag-and-drop para configurar posición de pantallas
- Configuración granular de servicios (mouse, teclado, clipboard, audio)
- Baja latencia optimizada para redes LAN
