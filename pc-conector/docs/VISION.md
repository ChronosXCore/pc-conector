# PC Conector - Vision

## 🌟 Visión del Proyecto

Convertir PC Conector en la herramienta de referencia para conectar PCs en red local, ofreciendo una experiencia tan fluida que el usuario sienta que opera una sola computadora con múltiples pantallas.

## 🚀 Metas a Largo Plazo

1. **Fase 1 - Fundación** ✅
   - Configurar proyecto Tauri con React + Rust
   - Establecer documentación y plan de desarrollo

2. **Fase 2 - Comunicación** 🔄
   - Implementar descubrimiento de peers en LAN (mDNS)
   - Establecer conexión P2P con cifrado
   - Sincronización de portapapeles en tiempo real

3. **Fase 3 - Control de Entrada** 📋
   - Captura y envío de eventos de mouse
   - Captura y envío de eventos de teclado
   - Integración de movimiento fluido entre pantallas

4. **Fase 4 - Audio** 🎵
   - Streaming de audio bidireccional
   - Selección de dispositivos de audio
   - Baja latencia optimizada

5. **Fase 5 - UI/UX** 🎨
   - Interfaz drag-and-drop para configuración de pantallas
   - Panel de control completo
   - Configuración de auto-inicio

6. **Fase 6 - Pulido** ✨
   - Pruebas exhaustivas en Windows y Linux
   - Optimización de rendimiento
   - Empaquetado y distribución

## 💡 Filosofía de Diseño

- **Configurable pero simple**: Que funcione out-of-the-box pero permita personalización total
- **Rendimiento primero**: Mínima latencia, máximo throughput
- **Multiplataforma**: Windows y Linux (Omarchy Linux) con el mismo código base
